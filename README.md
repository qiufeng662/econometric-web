# 一键实证分析 Web 应用

基于 Streamlit 构建的实证分析可视化工具，支持多种数据格式，提供完整的回归分析流程。

## 功能特性

- **数据支持**: .dta (Stata)、.xlsx/.xls (Excel)、.csv、.sav (SPSS)
- **变量识别**: 自动识别因变量、自变量、控制变量
- **分析方法**:
  - 描述性统计
  - 基准回归 (OLS + 稳健标准误)
  - 异质性分析 (分组回归)
  - 稳健性检验 (Winsorize、不同标准误)
  - 固定效应模型
  - 诊断检验 (VIF、BP 检验、DW 检验)
  - **显著性优化** (新增): 自动尝试多种方法优化回归结果，使核心变量显著
  - **计量经济学方法** (新增): 完整的因果推断和准实验设计方法
    - DID 方法：经典 DID、多期 DID、事件研究、DDD
    - 因果推断：工具变量法 (IV)、倾向得分匹配 (PSM)
    - 准实验设计：断点回归 (RDD)、合成控制法 (SCM)
    - 其他方法：分位数回归、面板数据模型
- **可视化**: 交互式图表展示
- **报告生成**: Word 格式分析报告（含显著性优化报告）

1. **安装依赖**

## 使用说明

### 1. 上传数据

支持格式：
- Stata 数据文件 (.dta) - **推荐**
- Excel 文件 (.xlsx, .xls)
- CSV 文件 (.csv)
- SPSS 数据文件 (.sav)

### 2. 变量选择

应用会自动识别变量类型：
- **因变量**: 结果变量 (Y)
- **核心自变量**: 处理变量 (X)
- **控制变量**: 协变量
- **固定效应**: 地区、时间等
- **分组变量**: 用于异质性分析
- **显著性优化**: 启用后自动尝试多种方法（Winsorize、固定效应、不同标准误等）优化结果
- **计量经济学方法**: 启用后可选择 DID、IV、PSM、RDD、SCM 等高级方法

### 3. 分析与下载

点击"开始分析"后，可以：
- 查看交互式图表
- 浏览回归结果表格
- 下载 Word 分析报告
- **显著性优化报告**（如启用优化）
- **计量经济学方法结果**（如启用）

### 4. 显著性优化功能

**使用方法**：
1. 在侧边栏勾选"启用显著性优化"
2. 选择目标显著性水平（0.01/0.05/0.1）
3. 点击"开始分析"
4. 在"显著性优化"标签页查看结果对比

**优化方法**：
- 数据层面：Winsorize 缩尾（1%/5%）、对数转换
- 模型层面：不同稳健标准误（HC1/HC2/HC3）、固定效应模型、聚类标准误
- 样本层面：剔除极端值
- 诊断检验：VIF、BP 检验、DW 检验、RESET 检验

⚠️ **学术诚信提醒**：本功能仅供学术研究和稳健性检验使用，请在学术诚信的前提下使用

### 5. 计量经济学方法（新增）

**支持的方法**：

#### DID 方法系列
- **经典 DID**: 适用于单一时点的政策评估
- **多期 DID**: 允许不同个体在不同时间接受处理
- **事件研究**: 分析政策动态效应，检验平行趋势
- **DDD**: 差异中的差异，控制额外异质性

#### 因果推断
- **工具变量法 (IV)**: 处理内生性问题，包含弱工具变量检验
- **倾向得分匹配 (PSM)**: 解决样本选择偏差，包含平衡性检验

#### 准实验设计
- **断点回归 (RDD)**: 利用断点进行因果识别
- **合成控制法 (SCM)**: 适用于小样本政策评估

#### 其他方法
- **分位数回归**: 分析不同分位点的异质性效应
- **面板数据模型**: 固定效应、随机效应模型

**使用方法**：
1. 在侧边栏勾选"启用计量经济学方法"
2. 选择具体方法或"自动检测"
3. 配置相应参数（如时间变量、断点值等）
4. 点击"开始分析"，结果在"计量经济学"标签页展示

---

## 注意事项

1. **数据格式**: 推荐使用 Stata (.dta) 格式，支持最好
2. **样本量**: 
   - 基准回归建议 N > 100
   - DID/PSM 等因果推断方法建议 N > 500
   - SCM 需要足够的时间序列数据
3. **缺失值**: 检查缺失比例，>30% 建议剔除
4. **中文路径**: 已支持中文文件路径
5. **计量方法假设**: 使用高级计量方法前，请确保理解其识别假设和适用条件
